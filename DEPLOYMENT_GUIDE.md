## ğŸ”§ Post-Deployment Steps

### 1. **Add Google Analytics**
Uncomment and configure the Google Analytics code in `index.html`:
- Create account at [analytics.google.com](https://analytics.google.com)
- Get your Measurement ID (e.g., `G-XXXXXXXXXX`)
- Replace `GA_MEASUREMENT_ID` in the code

### 2. **Update Sitemap URL**
In `robots.txt`, update the sitemap URL to your actual domain:
```
Sitemap: https://yourdomain.com/sitemap.xml
```

### 3. **Update Canonical URLs**
In all HTML files, update canonical URLs from:
```html
<link rel="canonical" href="https://www.husanbazar.com/">
```
To your actual domain.

### 4. **Submit to Search Engines**
- **Google Search Console**: [search.google.com/search-console](https://search.google.com/search-console)
  - Add your website
  - Submit your sitemap
  - Request indexing
  
- **Bing Webmaster Tools**: [bing.com/webmasters](https://www.bing.com/webmasters)
  - Add your site
  - Submit sitemap

### 5. **Apply for Google AdSense**
- Go to [google.com/adsense](https://www.google.com/adsense)
- Sign up with your website URL
- Add AdSense code to your website
- Wait for approval (typically 1-2 weeks)

### 6. **Set Custom Domain (Optional)**
If using GitHub Pages, Netlify, or Vercel:
- Purchase a domain (e.g., from Namecheap, GoDaddy)
- Configure DNS settings in your hosting dashboard
- Point domain to your hosting
